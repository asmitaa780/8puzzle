# 🧩 8 Puzzle Game (Web Version)

## 📌 Overview

This project is a **web-based implementation of the classic 8 Puzzle Game**, built using **HTML, CSS, and JavaScript**. The game consists of a 3×3 grid with 8 numbered tiles and one empty space. The objective is to rearrange the tiles to reach the correct order.

---

## 🎮 Features

* Interactive 3×3 puzzle board
* Tile movement using click
* Shuffle functionality
* Win condition detection
* Responsive UI design

---

## 🧠 Game Logic

* Only tiles adjacent to the empty space can move
* Each move slides a tile into the empty position
* The game is solved when tiles are arranged in order:

```
1 2 3
4 5 6
7 8 _
```
